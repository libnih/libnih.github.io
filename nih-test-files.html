TEST_FILENAME()

#define             TEST_FILENAME(_var)

Generate a filename that may be used for testing, it's unlinked if it exists and it's up to you to unlink it when done. _var should be at least PATH_MAX long.

_var :

 variable to store filename in.

TEST_FILE_EQ()

#define             TEST_FILE_EQ(_file, _line)

Check that the next line in the file _file is _line, which should include the terminating newline if one is expected.

_file :

 FILE to read from,

_line :

 line to expect.

TEST_FILE_MATCH()

#define             TEST_FILE_MATCH(_file, _pattern)

Check that the next line in the file _file matches the glob pattern _pattern, which should include the terminating newline if one is expected.

_file :

 FILE to read from,

_pattern :

 pattern to expect.

TEST_FILE_EQ_N()

#define             TEST_FILE_EQ_N(_file, _line)

Check that the start of the next line in the file _file is _line, up to the length of that argument.

_file :

 FILE to read from,

_line :

 line to expect.

TEST_FILE_NE()

#define             TEST_FILE_NE(_file, _line)

Check that the next line in the file _file is not _line, but also not end of file.

_file :

 FILE to read from,

_line :

 line to expect.

TEST_FILE_NE_N()

#define             TEST_FILE_NE_N(_file, _line)

Check that the next line in the file _file does not start with _line, up to the length of that argument; but also not end of file.

_file :

 FILE to read from,

_line :

 line to expect.

TEST_FILE_END()

#define             TEST_FILE_END(_file)

Check that the end of the file _file has been reached, and that there are no more lines to read.

_file :

 FILE to check.

TEST_FILE_RESET()

#define             TEST_FILE_RESET(_file)

This macro may be used to reset a temporary file such that it can be treated as a new one.

_file :

 FILE to reset.

TEST_EXPECTED_STR()

#define             TEST_EXPECTED_STR(_str, _filename)

Check that the string _str exactly matches the contents of the file _filename, which is local to the expected directory.

_str :

 string to check,

_filename :

 filename to compare against.

TEST_EXPECTED_FILE()

#define             TEST_EXPECTED_FILE(_file, _filename)

Check that the contents of file _file exactly matches the contents of the file _filename, which is local to the expected directory.

_file :

 open file to check,

_filename :

 filename to compare against.
